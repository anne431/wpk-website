###Aufgabe
Ihr Unternehmen beauftragt Sie und Ihre Gruppe eine Produktseite zu erstellen. 
Die Webseite soll mit Bootstrap erstellt werden.

Jedes Gruppenmitglied sollte das Tutorial (Bootstrap → Tutorial) durcharbeiten

Bevor eine Website gebaut wird, stehen Überlegungen an, wie sie aussehen und was sie beinhalten soll.

###Anforderungen
Die farbliche Gestaltung ist frei wählbar.
Die Webseite soll vorerst nur für Desktop PCs optimiert sein (responsive ist nicht gefordert).
Geben Sie der Webseite oder dem Unternehmen einen Namen. Seien Sie gerne kreativ.

Die Styles (CSS) sollen außerhalb der HTML Datei gespeichert und in die CSS eingebunden werden.

Die Webseite muss mindestens
- einen Header (z.B. Menü)
- einen Contentbereich (die Mitte)
- einen Footer beinhalten.

Die Webseite soll mindestens diese folgenden Menüs bzw. Seiten beinhalten:
- __Home__ z. B.: Darstellung der Produkte/Dienstleistungen und weiteres nach ihrem Wunsch
- __über uns__ z. B.: Darstellung der Gruppe
- __Impressum__ recherchieren Sie im Internet welche Angaben hier als Unternehmen notwendig sind

Schreiben Sie mindestens eine Javascript Funktion und binden Sie diese an einer beliebigen Stelle in der Webseite ein.

###Empfehlungen
Es wird von meiner Seite aus empfohlen, dass Sie auf Basis des "Bootstrap 4 Basic
Templates" arbeiten.

Bootstrap bietet viele vorgefertigte Komponenten, die Sie einsetzen können.
(z. B. https://getbootstrap.com/docs/4.0/components/jumbotron/)

###Abgabe:
- Bitte nennen Sie ggf. alle extern verwendeten Pakete(incl. Quelle) in einer Readme
Datei.
  
- Wo wurde auf der Webseite eine Javascript Funktion eingebettet? Bitte in der
Readme Datei nennen.

- Laden Sie hier ihr Projekt als Archiv (zip) gepackt hoch. 
  
  Der Dateiname des Archivs sollte die Namen der Gruppenmitglieder enthalten
